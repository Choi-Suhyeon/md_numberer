# MD_NUMBERER

Herein lies a program that provides numbering for each heading within a markdown file.
마크다운 파일 내의 각 제목에 대해 넘버링을 해주는 프로그램입니다.

## USEAGE GUIDELINES
`md_numberer --file | --directory <FILE_OR_DIRECTORY_PATH>`
